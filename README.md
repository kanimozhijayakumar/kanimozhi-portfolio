# Kanimozhi J - React Portfolio

A responsive professional portfolio converted from HTML/CSS/JS to a Vite + React app.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

## Build for production

```bash
npm run build
npm run preview
```

## Files

- `src/main.jsx` - React components and portfolio content
- `src/style.css` - Portfolio styling
- `public/kanimozhi_resume.pdf` - Resume file used by the Download Resume button
